# hetmers

Rust script for working with heterozygous k-mers for population genetics

# Usage

```
Usage: hetmers [OPTIONS] --input <INPUT> --output <OUTPUT>

Options:
  -i, --input <INPUT>
  -m, --minimum <MINIMUM>  [default: 1]
  -a, --alleles <ALLELES>  [default: 2]
  -o, --output <OUTPUT>
  -h, --help               Print help
  -V, --version            Print version
```
